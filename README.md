# 🌱 Agrinho 2026 - Agronegócio

Este é um projeto educacional desenvolvido para o **Agrinho 2026**, com o tema **Agronegócio**.  
O site tem como objetivo explicar de forma simples como o campo e a cidade estão conectados através da produção agrícola, tecnologia e economia.

---

## 📁 Estrutura do Projeto

O projeto é composto pelos seguintes arquivos:
📦 agrinho-2026
┣ 📄 index.html → Estrutura principal do site
┣ 🎨 style.css → Estilo visual da página
┣ ⚙️ script.js → Interatividade (botões, quiz e curiosidades)
┗ 📄 README.md → Documentação do projeto

---

## 🚀 Como executar o projeto

1. Baixe ou copie todos os arquivos para uma mesma pasta
2. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox)
3. Navegue pelo site usando os botões do menu

---

## 🌾 Funcionalidades

- Página inicial explicando o agronegócio
- Seção sobre a importância do setor
- Informações sobre tecnologia no campo
- Quiz interativo com respostas
- Curiosidades aleatórias sobre o agronegócio

---

## 💻 Tecnologias utilizadas

- HTML5 → Estrutura do site
- CSS3 → Estilização e layout
- JavaScript → Interatividade

---

## 🎯 Objetivo

Mostrar a importância do agronegócio para a economia, alimentação e tecnologia, aproximando o conhecimento do campo com a vida urbana.

---

## 👨‍🎓 Projeto escolar

Desenvolvido como atividade do programa **Agrinho 2026**, incentivando a aprendizagem sobre o meio rural e sua importância para a sociedade.
